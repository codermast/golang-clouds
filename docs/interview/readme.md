## 面试技巧

### 回答框架

```
1. 是什么（定义）
2. 为什么（原理/原因）
3. 怎么做（实践/代码）
4. 延伸（对比/扩展）
```

### 面试准备

- **基础扎实**：数据结构、算法、计算机基础
- **项目深挖**：准备 2-3 个能展示技术深度的项目
- **源码阅读**：核心框架的关键实现
- **实战经验**：遇到的问题和解决方案
