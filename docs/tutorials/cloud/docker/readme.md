# Docker

Docker 是一个开源的容器化平台，可以让开发者打包应用及其依赖到一个可移植的容器中，实现"一次构建，到处运行"。

## 为什么要学 Docker？

众所周知，Docker 的教程网络上非常多，也不乏有许多大佬整理和总结的优秀笔记。但是对于 Java 后端开发人员来说，目前市面上的 Docker 教程要么过于繁琐，要么过于精炼。过于繁琐学起来费时费力，过于精炼又不能达到快速掌握 Docker 核心的目的。

本系列旨在帮助 Java 后端开发者快速掌握 Docker 核心知识，记录学习心得和笔记。
