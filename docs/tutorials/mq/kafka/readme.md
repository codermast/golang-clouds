# Kafka

Apache Kafka 是一个分布式流处理平台，最初由 LinkedIn 开发，后成为 Apache 顶级项目。Kafka 以高吞吐量、可扩展性和持久性著称，广泛应用于日志收集、流处理、事件驱动架构等场景。
