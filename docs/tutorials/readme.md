## 推荐学习顺序

### Java 技术栈

```
1. Java 核心 → 2. 集合框架 → 3. IO/多线程 → 4. JVM
      ↓
5. MySQL → 6. Spring → 7. SpringMVC → 8. SpringBoot
      ↓
9. Redis → 10. 消息队列 → 11. SpringCloud
      ↓
12. Docker → 13. Kubernetes → 14. 项目实战
```

### Go 技术栈

```
1. Go 基础语法 → 2. 复合类型 → 3. 接口
      ↓
4. 并发编程 → 5. 标准库 → 6. Web 框架（Gin/Echo）
      ↓
7. 数据库操作 → 8. 微服务 → 9. 云原生开发
```
